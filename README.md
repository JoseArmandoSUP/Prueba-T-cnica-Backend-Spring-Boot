# Prueba Técnica – Backend Spring Boot

INSTRUCCIONES PARA CLONAR EL PROYECTO:

1. Copie la url del repositorio

2. Abra un terminal, situece en el directorio donde desee clonar el repositorio y ejecute el commando:  _"git clone <*url del repositotio*>"_



INTRUCCIONES PARA EJECUTAR EL PROYECTO:

1. Una vez clonado el repositorio, dentro de la terminal, situece en la carpeta "backend" _"cd backend"_, ya dentro de la carpeta ejecute el comando _"pnpm install"_, este comando descargara todas als dependencias que contenga el modulo

2. Repita el proceso con la carpeta "frontend"
